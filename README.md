# ThunderFighter 雷霆战机

基于 Java Swing 开发的纵版飞机射击游戏。玩家控制战机移动、自动发射子弹、击败敌机和 Boss，并通过拾取道具提升战斗能力。

## 项目简介

本项目是面向 Java 面向对象程序设计课程的综合实训作品，主要实现了游戏主界面、敌机生成、碰撞检测、爆炸动画、道具奖励、Boss 关卡、用户登录注册、积分存储和皮肤切换等功能。

## 主要功能

- 用户注册与登录
- 游戏得分与用户积分存储
- 积分解锁并装备战机皮肤
- 战机双帧尾焰动画
- 敌机与 Boss 战斗系统
- 子弹碰撞与爆炸动画
- 小蜜蜂、双倍火力、核弹奖励道具
- 道具 Z 字形下落
- 生命值上限控制
- 关卡推进与胜利/失败界面
- 右侧信息面板与雷达显示

## 操作说明

| 按键 | 功能 |
| --- | --- |
| W / A / S / D | 移动战机 |
| 方向键 | 移动战机 |
| H | 使用核弹 |
| U / Enter | 开始游戏 |
| I | 暂停游戏 |
| O | 继续游戏 |
| Q | 退出游戏 |

## 运行环境

- JDK 8 或以上版本
- Windows / macOS / Linux 均可运行
- 推荐使用 IntelliJ IDEA 打开项目
